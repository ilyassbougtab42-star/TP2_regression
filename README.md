# TP2 : Régression linéaire et logistique

## Objectifs
- Prédire la consommation de carburant (MPG) avec régression linéaire.
- Prédire l’admission d’étudiants avec régression logistique.

## Étapes
1. Chargement et nettoyage des données.
2. Visualisation et analyse.
3. Entraînement des modèles.
4. Évaluation des performances.

## Résultats
- Régression linéaire : précision ~0.84
- Régression logistique : précision ~0.74

## Contenu du projet
- `TP2_regression.ipynb` : notebook Jupyter
- `auto-mpg.csv` : dataset Auto MPG
- `binary.csv` : dataset UCLA
